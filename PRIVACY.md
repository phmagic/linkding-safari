# Privacy Policy

_Last updated: 2026-04-24_

**linkding for Safari** is free and open source software. It does not collect,
transmit, or sell any personal data to the developer or to any third party.

## What the extension stores

The extension saves your preferences using Safari's extension storage:

- The URL of your self-hosted linkding server
- Your linkding API token
- Default options you choose (tags, unread, archive, sharing)

These values are used only to communicate with the linkding server that **you**
configure.

## What the extension sends over the network

The only network requests the extension makes go to the linkding server URL
that you enter in settings. These requests are used to save bookmarks, check
whether a page is already bookmarked, and search your bookmarks. No requests
are sent to the developer or to any other third party.

## iCloud sync

Safari uses `storage.sync`, which means your saved preferences (including the
server URL and API token) may sync across your own Apple devices via iCloud
when iCloud sync for Safari extensions is enabled. This data stays within your
own Apple account; the developer has no access to it.

## Analytics and tracking

None. The extension contains no analytics, telemetry, advertising, or tracking
code.
